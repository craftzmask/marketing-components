# Profile Card

A static profile card UI built from a Figma design. The layout focuses on clean semantics, reusable class naming, and accessible interactions.

## Screenshot

![Result Screenshot](/profile-card/img/result.png)

## Built With

- HTML5
- CSS3 (Flexbox + modern CSS nesting)
- Google Fonts (Noto Sans)
- Inline SVG icons

## Features

- Semantic structure (`main`, `article`, `header`, `footer`)
- BEM-style class naming for maintainable styles
- Keyboard-friendly focus states (`:focus-visible`)
- Icon links using SVG + `currentColor`
- Circular avatar styling
