# Testimonial Card (HTML + CSS)

A small UI challenge built with **only HTML and CSS**.  
The goal is to practice **semantic HTML**, **Flexbox layout**, and **card styling** (spacing, border radius, shadow, typography).

## Screenshot

![Result Screenshot](/testimonial-card/img/result.png)

## Built With

- HTML5
- CSS3
- Google Fonts: Noto Sans

## What I Practiced

- Using **semantic HTML** for meaningful structure:
  - `figure` for the card container
  - `figcaption` for author info
  - `blockquote` for the testimonial text
- **Flexbox** for aligning the avatar and author text
- Card UI styling:
  - padding + spacing (`gap`)
  - border radius
  - box shadow
- Typography:
  - font sizing and line-height
  - color contrast for readable text
